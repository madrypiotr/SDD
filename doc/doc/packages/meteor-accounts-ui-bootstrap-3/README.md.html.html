meteor-accounts-ui-bootstrap-3
  </h1>
</div>
<p>Meteor accounts-ui styled with twitter/bootstrap 3</p>
<div class="pilwrap" id="prerequisites">
  <h2>
    <a href="#prerequisites" name="prerequisites" class="pilcrow"></a>
Prerequisites
  </h2>
</div>
<p>Use the meteorite package manager
http://oortcloud.github.com/meteorite/</p>
<p><code>[sudo] npm install -g meteorite</code></p>
<p>You will also need the following:</p>
<ul>
<li>Bootstrap 3: <code>mrt add bootstrap-3</code></li>
<li>At least one accounts plugin: <code>mrt add accounts-password</code>, <code>mrt add accounts-github</code>, etc.</li>
</ul>
<div class="pilwrap" id="how-to-add-to-your-meteor-app">
  <h2>
    <a href="#how-to-add-to-your-meteor-app" name="how-to-add-to-your-meteor-app" class="pilcrow"></a>
How to add to your meteor app
  </h2>
</div>
<p><code>mrt add accounts-ui-bootstrap-3</code></p>

<div class="pilwrap" id="add-additional-logged-in-actions">
  <h2>
    <a href="#add-additional-logged-in-actions" name="add-additional-logged-in-actions" class="pilcrow"></a>
Add additional logged in actions
  </h2>
</div>
<p>You can add additional markup to the logged in dropdown, e.g. to edit
the user's account or profile, by defining a
<code>_loginButtonsAdditionalLoggedInDropdownActions</code> template and specifying
the corresponding events.</p>
<pre><code class="html"><span class="hljs-tag">&lt;<span class="hljs-name">template</span> <span class="hljs-attr">name</span>=<span class="hljs-string">"_loginButtonsAdditionalLoggedInDropdownActions"</span>&gt;</span>
	<span class="hljs-tag">&lt;<span class="hljs-name">button</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"btn btn-default btn-block"</span> <span class="hljs-attr">id</span>=<span class="hljs-string">"login-buttons-edit-profile"</span>&gt;</span>Edytuj profil<span class="hljs-tag">&lt;/<span class="hljs-name">button</span>&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-name">template</span>&gt;</span>
</code></pre>
<pre><code class="javascript">Template._loginButtonsLoggedInDropdown.events({
	<span class="hljs-string">'click #login-buttons-edit-profile'</span>: <span class="hljs-function"><span class="hljs-keyword">function</span>(<span class="hljs-params">event</span>) </span>{
		event.stopPropagation();
		Template._loginButtons.toggleDropdown();
		Router.go(<span class="hljs-string">'profileEdit'</span>);
	}
});
</code></pre>
